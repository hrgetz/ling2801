## Course Description
This is an introductory course on language and the brain, designed for students with no background in neuroscience. Its goal is to introduce students to the major scientific findings in this area as well as the scientific process that generated this evidence. Topics include the brain regions and networks involved in language processing in the healthy brain; how these structures develop in children; and how language skills are affected by damage to these structures.

## Learning Goals
The course is designed to achieve the following learning outcomes:
1. Knowledge of the basic principles, foundational evidence, and current research challenges in the field of language and the brain
2. Understanding of the scientific process through which researchers learn about this field
3. The ability to understand and interpret scientific evidence about language in the brain
4. An understanding of how research on language and the brain relates to broader societal issues
